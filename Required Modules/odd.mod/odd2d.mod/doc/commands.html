Function SetBorderColor( red:Int, green:Int, blue:Int )
DescriptionSet current border color.
InformationThe red, green and blue parameters should be in the range of 0 to 255.

The default border color is black.


Function SetBorderMode( mode:Int )
DescriptionSets how borders displayed.
InformationSets how borders will be displayed if the virtual aspect ratio is not the same as the screens aspect ratio.

The available modes are:
BORDER_NONEStretches/distorts the display to fit the screen. This is the default Max2D behaviour
BORDER_LETTERBOX_FILLScales the display to fit the screen but maintains it's aspect ratio. Any draw space left around it is not clipped
BORDER_LETTERBOX_SOLIDScales the display to fit the screen but maintains it's aspect ratio. Any draw space left around it is clipped
BORDER_BESTFIT_FILLScales the display by whole numbers only whilst maintaining it's aspect ratio. Any draw space left around it is not clipped
BORDER_BESTFIT_SOLIDScales the display by whole numbers only whilst maintaining it's aspect ratio. Any draw space left around it is clipped


Function SetMidFocus()
DescriptionSets the screen focus to the center of the display.

Function SetScreenFocus( x:Float, y:Float )
DescriptionSet display focus.
InformationThe screen focus is a 2D offset added to the x,y location of all drawing commands. It is applied after all other transforms and can be thought of as the screens handle.

In Max2D the focus is allways the top lext of the screen meaning all other cammands ultimately take their origin from that, SetScreenFocus allows you to change this behaviour.


Function SetScreenRotation( rot:Float )
DescriptionSet screen rotation.
Informationrotation is given in degrees and should be in the range 0 to 360.

SetScreenRotation rotates the whole display around the focus.


Function SetZoom( zoom:Float )
DescriptionSet current screen zoom.
InformationZooms the display in or out centered upon the focus.
